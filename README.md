<h2 align="center">
Element to Tuple Prompting (E2TP)
</h2>

### ✨ Introduction 
This implementation is for a new prompting strategy applied to the task of aspect-sentiment tuple prediction, which utilizes a two‑step procedure—under a data‑efficient diet paradigm and two non‑diet ones using data augmentation—in which the first step predicts single elements independently, while the second maps these predicted elements to their corresponding tuples.
